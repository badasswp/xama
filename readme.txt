# Xama

## Description

A simple WP plugin to help you set up Quizzes.

## Installation

1. **WordPress Dashboard Installation:**
   - Navigate to the WordPress admin dashboard.
   - Go to `Plugins > Add New`.
   - Search for "Xama".
   - Click `Install Now` and then `Activate`.

2. **Manual Installation:**
   - Download the plugin zip file from [Your Plugin Repository/Website].
   - Extract the zip file contents.
   - Upload the plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
   - Activate the plugin through the 'Plugins' menu in WordPress.
